# Unity 语义地图数据说明

本目录数据来源于本地 Unity 语义地图标注工程导出文件，用于说明自动驾驶语义地图标注与导航数据准备流程。

## 文件说明

- `lane.csv`：车道编号、前后继车道、限速、车道类型等拓扑信息。
- `dtlane.csv`：车道中心线采样点、距离、方向、坡度等属性。
- `roadedge.csv`：道路边界或路沿线关联关系。
- `auto.pcd`：点云地图数据，可用于点云地图加载、语义标注或后续感知/导航实验。

## 注意事项

- CSV 字段格式接近 Autoware/vector map 的课程实践数据格式。
- 该目录只保留用于作品集展示和复现实验的数据，不包含完整 Unity 工程。
- 如需在 ROS 中使用，需要根据实际工具链编写转换或加载脚本。
